Проект №1

Техническое задание на проект "To Do List"

Цель проекта:
Разработка веб-приложения "To Do List" с использованием Node.js и технологий Express, Multer, JWT, Bcrypt и MySQL2 (Promis). Приложение должно обеспечивать авторизацию пользователей, создание записей, загрузку изображений и реализацию системы напоминаний через оповещения на сайте.

Авторизация пользователей:
Регистрация нового пользователя с использованием Bcrypt для хеширования паролей.
Вход в систему с использованием JWT.
Возможность выхода из системы.
Форма регистрации и входа в систему.
Хранение токена JWT в локальном хранилище для последующей аутентификации запросов.

Создание, чтение, обновление и удаление записей.

Отображение списка записей с возможностью создания, редактирования и удаления записей.

Использование Multer для обработки загрузки изображений.
Ограничение на размер загружаемого файла и типы файлов (например, только изображения).
Форма для загрузки изображений при создании или редактировании записи.
Отображение загруженных изображений в списке записей.

*Система напоминаний:
Возможность добавления напоминаний к записям.
Отображение уведомлений о предстоящих напоминаниях на главной странице.

Маршрутизация:
Использование React Router DOM для навигации между страницами (например, главная страница, страница создания записи, страница входа и регистрации).

Веб-приложения "To Do List" необходимо реализовать следующие страницы:

Страница регистрации:
Форма для создания нового пользователя.
Поля для ввода имени, электронной почты и пароля.
Кнопка для отправки данных на сервер.

Страница входа:
Форма для авторизации пользователя.
Поля для ввода электронной почты и пароля.
Кнопка для входа в систему.

Главная страница (Список задач):
Отображение всех записей (задач) пользователя.
Кнопка для создания новой записи.
Возможность редактирования и удаления существующих записей.
Отображение напоминаний и уведомлений.

Страница создания/редактирования записи:
Форма для добавления новой записи или редактирования существующей.
Поля для ввода текстового описания задачи и загрузки изображения.
Кнопка для сохранения изменений.

Страница ошибок (404):
Сообщение о том, что страница не найдена.
Ссылка для возврата на главную страницу

Веб-приложения "To Do List" необходимые  endpoints:

Авторизация пользователей:

POST /api/auth/register
Регистрация нового пользователя. Принимает данные (имя, электронная почта, пароль) и создает нового пользователя.
POST /api/auth/login
Вход в систему. Принимает электронную почту и пароль, возвращает JWT токен для аутентификации.

Управление записями (задачами):
GET /api/tasks/user/:id
Получение списка всех задач текущего пользователя. Возвращает массив задач.

POST /api/tasks/user/:id
Создание новой задачи. Принимает данные (описание задачи, изображение) и создает новую запись.

GET /api/tasks/:id
Получение конкретной задачи по ID. Возвращает данные задачи.

PUT /api/tasks/:id
Обновление существующей задачи. Принимает данные (описание задачи, изображение) и обновляет запись.

DELETE /api/tasks/:id
Удаление задачи по ID. Удаляет запись из базы данных.
